# Система технической поддержки

Две версии системы технической поддержки на Python:

## Файлы проекта

### `main.py` 
**Административная версия** - полный контроль над тикетами:
- Просмотр всех заявок
- Изменение статусов
- Статистика и поиск
- Внутренние комментарии

### `app.py`
**Пользовательская версия** - простой интерфейс:
- Создание новых тикетов
- Просмотр своих заявок
- Добавление комментариев
- Отслеживание статусов
